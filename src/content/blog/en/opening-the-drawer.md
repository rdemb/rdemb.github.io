I have a drawer. Metaphorical, though it is really a few folders on a disk, full of projects I once started and set aside. Each had the same fate: a spark, a few evenings, then a new idea that looked more interesting.

At some point I decided this was a bad trade. A new idea always looks more attractive than finishing an old one, because the new one is still clean. It has no boring details, no bugs, no fixes. And yet value only appears when you close something.

So I opened the drawer. GAMBIT and Order Flow are not new ideas. They are old ideas I finally brought to a state where I can show them to someone.

I learned one thing along the way. Finishing teaches more than starting. Starting is a pleasure, finishing is a craft, and only the craft really stays.
