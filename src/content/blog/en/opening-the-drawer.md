I have a drawer — metaphorical, though it's really a few folders on a disk — full of projects I once started and set aside. Each had the same fate: a spark, a few evenings, then a new idea that looked more interesting.

At some point I decided this was a bad trade. A new idea always looks more attractive than finishing an old one, because the new one is still clean — no boring details, no bugs, no fixes. But value only appears in the closing.

So I opened the drawer. GAMBIT and Order Flow are not new ideas. They're old ideas I finally brought to a state where I can show them to someone.

I learned one thing along the way: finishing teaches more than starting. Starting is pleasure, closing is craft — and only the craft stays.
