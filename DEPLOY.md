# Deploy Notes

This repository deploys to GitHub Pages via GitHub Actions on push to `main`.

## Local Setup

Deploy credentials are managed locally outside this repository.
See `SECURITY.md` for what must never be committed.

## Content Workflow

Posts: `_posts/pl`, `_posts/en`, `_posts/de`
Pages: root + `en/` + `de/`
Navigation: `_data/navigation.yml`
